# 🏗️ Projets Python

Ce dossier contient des projets complets pour mettre en pratique tout ce que tu as appris.

### Idées de projets à réaliser :
1. **Scraper Web** : Utilise `requests` et `BeautifulSoup` pour extraire les prix d'un site.
2. **Bot Discord** : Crée un compagnon pour ton serveur.
3. **Gestionnaire de mots de passe** : Un outil en ligne de commande pour stocker tes accès en toute sécurité.
4. **Analyseur de logs** : Un script qui lit des fichiers texte et génère un rapport d'erreurs.

> 💡 Chaque projet devrait avoir son propre sous-dossier avec un fichier `requirements.txt` dédié.
